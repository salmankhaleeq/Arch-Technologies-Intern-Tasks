# Customer Segmentation using Machine Learning

## Overview
This project implements an end-to-end customer segmentation pipeline using K-Means clustering.

## Features
- Data preprocessing
- Feature scaling
- K-Means clustering
- Model evaluation using Silhouette Score
- Interactive dashboard using Streamlit

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Streamlit

## How to Run

```bash
pip install -r requirements.txt
python main.py
streamlit run dashboard/app.py
